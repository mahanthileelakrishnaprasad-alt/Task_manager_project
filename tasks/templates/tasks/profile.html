How reminders work
  • Tasks: set a one-time reminder date & time when adding or editing a task.
  • Daily Routines: set a daily reminder time — it fires once per day until you tick the task off.
  • Reminders are checked periodically in the background, so there can be a short delay (a few minutes) between the set time and the email arriving.
